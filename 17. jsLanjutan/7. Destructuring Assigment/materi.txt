Materi: Destructuring Assignment di JavaScript

Destructuring Assignment adalah sebuah fitur dalam JavaScript yang memungkinkan kita untuk "mendestruksi" (mengambil) nilai-nilai dari array atau objek dan menyimpannya ke dalam variabel terpisah. Dengan menggunakan Destructuring Assignment, kita dapat mengakses dan menggunakan nilai-nilai tersebut dengan lebih mudah.

Destructuring Assignment pada Array:
Dalam Destructuring Assignment pada array, kita dapat menentukan variabel-variabel yang akan menerima nilai-nilai dari array tersebut dengan menggunakan pola yang mirip dengan array itu sendiri.

Contoh:

```javascript
// Deklarasi array
const colors = ['merah', 'kuning', 'hijau'];

// Destructuring Assignment
const [warna1, warna2, warna3] = colors;

// Menggunakan nilai-nilai yang didapatkan
console.log(warna1); // Output: 'merah'
console.log(warna2); // Output: 'kuning'
console.log(warna3); // Output: 'hijau'
```

Dalam contoh di atas, kita mendestruksi array `colors` menjadi tiga variabel: `warna1`, `warna2`, dan `warna3`. Setiap variabel akan menerima nilai yang sesuai dari array tersebut. Dengan begitu, kita dapat mengakses nilai-nilai array dengan menggunakan variabel-variabel tersebut.

Destructuring Assignment pada Objek:
Dalam Destructuring Assignment pada objek, kita dapat menentukan variabel-variabel yang akan menerima nilai-nilai dari properti-properti objek dengan menggunakan nama properti yang sesuai.

Contoh:

```javascript
// Deklarasi objek
const person = {
  name: 'John Doe',
  age: 30,
  city: 'Jakarta'
};

// Destructuring Assignment
const { name, age, city } = person;

// Menggunakan nilai-nilai yang didapatkan
console.log(name); // Output: 'John Doe'
console.log(age); // Output: 30
console.log(city); // Output: 'Jakarta'
```

Dalam contoh di atas, kita mendestruksi objek `person` menjadi tiga variabel: `name`, `age`, dan `city`. Setiap variabel akan menerima nilai yang sesuai dari properti-properti objek tersebut. Dengan begitu, kita dapat mengakses nilai-nilai objek dengan menggunakan variabel-variabel tersebut.

Destructuring Assignment dengan Default Value:
Dalam Destructuring Assignment, kita juga dapat menentukan nilai default untuk variabel-variabel yang mendestruksi nilai-nilai dari array atau objek. Nilai default akan digunakan jika nilai yang sesuai tidak ada atau undefined.

Contoh:

```javascript
// Deklarasi array
const numbers = [1, 2];

// Destructuring Assignment dengan default value
const [a, b, c = 3] = numbers;

// Menggunakan nilai-nilai yang didapatkan
console.log(a); // Output: 1
console.log(b); // Output: 2
console.log(c); // Output: 3
```

Dalam contoh di atas, kita mendestruksi array `numbers` menjadi tiga variabel: `a`, `b`, dan `c`. Variabel `c` memiliki default value 3. Karena `numbers` hanya memiliki dua nilai, maka `c`

 akan menggunakan default value-nya.

Cara Implementasi Destructuring Assignment:
1. Deklarasikan array atau objek yang akan didestruksi.
2. Tentukan variabel-variabel yang akan menerima nilai-nilai dari array atau objek tersebut.
3. Gunakan pola yang sesuai dalam Destructuring Assignment.
4. Variabel-variabel tersebut sekarang berisi nilai-nilai yang didapatkan dari array atau objek.
5. Gunakan variabel-variabel tersebut sesuai kebutuhan.

Contoh Implementasi:

```javascript
// Destructuring Assignment pada array
const numbers = [1, 2, 3];
const [x, y, z] = numbers;
console.log(x); // Output: 1
console.log(y); // Output: 2
console.log(z); // Output: 3

// Destructuring Assignment pada objek
const person = {
  name: 'Jane Doe',
  age: 25,
  city: 'Surabaya'
};
const { name, age, city } = person;
console.log(name); // Output: 'Jane Doe'
console.log(age); // Output: 25
console.log(city); // Output: 'Surabaya'
```

Dalam contoh di atas, kita mendestruksi array `numbers` dan objek `person` untuk mendapatkan nilai-nilai yang sesuai. Setelah mendestruksi, kita dapat menggunakan variabel-variabel `x`, `y`, `z`, `name`, `age`, dan `city` sesuai kebutuhan.
