1. CSS Property Filter:
   CSS Property Filter digunakan untuk memberikan efek visual pada elemen HTML, seperti penyesuaian warna, kecerahan, kontras, dan banyak lagi. Properti filter memungkinkan Anda untuk mengubah tampilan elemen dengan menerapkan filter visual pada gambar, latar belakang, atau elemen lainnya.

   Beberapa nilai properti filter yang umum digunakan meliputi:
   - `grayscale()`: Mengubah gambar menjadi skala abu-abu.
   - `blur()`: Memberikan efek blur (kabur) pada elemen.
   - `brightness()`: Menyesuaikan kecerahan elemen.
   - `contrast()`: Menyesuaikan kontras elemen.
   - `sepia()`: Memberikan efek sepia pada gambar.
   - `hue-rotate()`: Mengubah hue (warna) elemen dengan rotasi.

   Contoh penggunaan CSS Property Filter:
   ```css
   /* Memberikan efek filter pada gambar */
   .image {
     filter: grayscale(100%) blur(5px);
   }
   ```

   Dalam contoh di atas, elemen dengan kelas "image" akan diubah menjadi skala abu-abu sepenuhnya dan diberikan efek blur sebesar 5 piksel.

2. Cara Penggunaan:
   Untuk menggunakan properti filter, Anda perlu menyebutkan properti tersebut pada elemen target dalam file CSS. Anda dapat menggabungkan beberapa filter dalam satu deklarasi filter menggunakan spasi.

   Contoh penggunaan multiple filter:
   ```css
   /* Menggabungkan beberapa filter */
   .image {
     filter: grayscale(100%) blur(5px) brightness(120%);
   }
   ```

   Anda juga dapat menggabungkan properti filter dengan transformasi lainnya, seperti properti transform yang telah dibahas sebelumnya.

   Contoh penggunaan filter dengan transform:
   ```css
   /* Mengkombinasikan filter dan transform */
   .image {
     width: 200px;
     height: 200px;
     background-image: url('gambar.jpg');
     filter: grayscale(100%);
     transform: scale(1.2);
   }
   ```

   Dalam contoh di atas, elemen dengan kelas "image" akan ditampilkan sebagai gambar dengan skala abu-abu sepenuhnya dan diperbesar 1,2 kali ukuran aslinya.
